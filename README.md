## 使用说明

### 本地预览
直接双击 `index.html` 文件在浏览器中打开即可查看简历。

### 部署到 GitHub Pages
1. 创建新仓库
2. 上传所有文件
3. 进入 Settings → Pages
4. Source 选择 `main` 分支和 `/(root)`
5. 保存后即可通过 `https://你的用户名.github.io/仓库名` 访问
